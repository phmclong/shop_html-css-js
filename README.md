# Mục tiêu: 
- Clone lại web Shopee
- Sử dụng được HTML, CSS (áp dụng được các kĩ thuật như Grid, Responsive, Flexbox...)
- Sử dụng được JavaScript thuần (Vanila JavaScript) để thực hiện thao tác trên trình duyệt
